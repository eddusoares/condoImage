Teste de Configuração BASE_URL

URLs que devem funcionar:

Desenvolvimento (localhost):

  • Home
  • Condo Building
  • Neighborhood
  • Search

Produção (staging):

  • Home
  • Condo Building
  • Neighborhood
  • Search

Teste JavaScript:

Para testar, abra o console do navegador (F12) e execute:

// Verificar se BASE_URL está definido corretamente
console.log('BASE_URL:', window.BASE_URL);
console.log('APP_URL:', window.APP_URL);

// Testar construção de URL
const testEndpoint = 'condo-building';
const testUrl = new URL(testEndpoint);
console.log('URL construída:', testUrl.toString());

Como verificar o problema:

  1. Abra a página que contém os building cards
  2. Abra o console do navegador (F12)
  3. Clique no botão "Reset" da busca
  4. Verifique no console ou na aba Network qual URL está sendo chamada
  5. Deve ser: http://localhost:8000/condo-building?limit=6 ou https://condoimage.com/staging/condo-building?limit=6

Configuração .env para cada ambiente:

Desenvolvimento (local):

APP_URL=http://localhost:8000
BASE_URL=http://localhost:8000

Produção (staging):

APP_URL=https://condoimage.com
BASE_URL=https://condoimage.com/staging
